# GAgentManager

企业级Agent管理平台

## 项目结构

此项目采用模块化架构，主要分为四个部分：

### 模块说明

- `docs/` - 文档文件夹，包含项目的所有技术文档和使用说明
- `frontend/` - 前端代码文件夹，包含管理端和用户端界面
  - `admin/` - 管理端界面，提供管理员界面用于Agent管理和系统监控
  - `user/` - 用户端界面，提供最终用户使用的前端界面
- `backend/` - 后端服务代码文件夹，包含API服务和业务逻辑实现

## 快速开始

1. 各模块独立开发，可根据需求分别启动
2. 详细开发指南参见对应模块的README文件
