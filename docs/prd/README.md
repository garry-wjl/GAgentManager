此目录包含GAgentManager项目的所有产品需求文档。

## 版本说明

## 内容概要

PRD文档包含以下核心功能：

1. **Agent管理** - 管理企业内部所有的AI Agent
2. **用户管理** - 控制用户访问权限和管理用户
3. **模型管理** - 管理AI模型的生命周期
4. **Skill商店** - 提供Skill的发现、安装、更新和管理
5. **MCP管理** - 管理Model Context Protocol服务和连接
6. **工作流管理** - 提供可视化的工作流设计和管理
7. **监控与告警** - 实时监控系统和Agent运行状态
8. **首页与仪表盘** - 个性化信息展示和快速操作入口

所有功能都已明确定义了用户故事、具体需求和验收标准。
