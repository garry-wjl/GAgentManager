# 文档

此目录包含GAgentManager项目的所有文档。

## 目录结构

根据项目文档管理规则，文档按以下分类组织：

- `prd/` - 产品需求文档 (Product Requirements Documents)
- `技术方案/` - 技术方案文档 (Technical Specifications)
- `测试方案/` - 测试方案文档 (Testing Specifications)
- `issue/` - 规划文件 (Planning Documents)
- `other/` - 其他零散文件 (Miscellaneous Files)
- `api/` - API文档
- `architecture/` - 架构设计文档
- `tutorials/` - 教程文档
- `guides/` - 使用指南

## 文档管理规则

1. **PRD文档**：所有产品需求文档必须放在 `prd/` 目录下
2. **技术方案文档**：所有技术方案文档必须放在 `技术方案/` 目录下
3. **测试方案文档**：所有测试方案文档必须放在 `测试方案/` 目录下
4. **规划文件**：所有规划、任务、路线图等文件必须放在 `issue/` 目录下
5. **其他文件**：临时、测试或其他不适合分类的文件放在 `other/` 目录下

请遵循这些规则，以便更好地管理项目文档。
